## Prompt Log 2

**场景**：生成写作模板

**我的提示词**：
```
请为大学英语四级写作生成5个万能模板：

1. 议论文开头模板
2. 对比观点模板
3. 解决问题模板
4. 图表描述模板
5. 书信写作模板

每个模板包含：
- 适用场景说明
- 英文模板句
- 中文翻译
- 使用示例
```

**AI响应**：
```
## 1. 议论文开头模板

**适用场景**：讨论某个现象的利弊

**模板句**：
In recent years, [现象] has become a hot topic. 
While some people believe that [观点A], others argue that [观点B].
As far as I am concerned, I am in favor of [我的观点].

**中文翻译**：
近年来，[现象]已成为热门话题。
有些人认为[观点A]，而另一些人则认为[观点B]。
就我而言，我支持[我的观点]。
```

**效果评估**：
- ✅ 满意
- 改进方向：可增加高级词汇替换建议

## Prompt Log 3

**场景**：生成课堂讨论问题

**我的提示词**：
```
请为大学英语精读课文生成8张课堂讨论卡。
每张卡片包含：
1. 一个开放式问题
2. 2-3个追问提示
3. 预期回答方向

问题类型包括：事实理解、观点表达、文化对比、批判性思考
```

**AI响应**：
```
## Discussion Card 1: 事实理解

**问题**：What is the main argument of the passage?

**追问提示**：
- Can you find the thesis statement?
- How does the author support the main point?

**预期回答**：
The main argument is that technology has changed education.
The thesis appears in the second paragraph.
```

**效果评估**：
- ✅ 满意
- 改进方向：可根据不同英语水平调整问题难度

## 提示词技巧总结

根据今天的练习，总结你学到的提示词技巧：

1. **具体化**：越具体的指令，AI响应越准确
   - 指定输出格式（Markdown表格、列表等）
   - 明确字数要求
   - 给出示例

2. **结构化**：使用清晰的结构（如：主题+要求+格式）
   - 使用编号分隔不同需求
   - 用粗体强调关键词
   - 先说明场景，再给出指令

3. **迭代优化**：根据AI响应不断调整提示词
   - 第一轮获取基础内容
   - 第二轮补充细节
   - 第三轮优化格式

4. **角色设定**：为AI设定一个身份会提高专业性
   - "你是一位大学英语教师"
   - "你是一位四级写作考官"
