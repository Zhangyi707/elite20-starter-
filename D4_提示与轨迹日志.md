### 轨迹2：Python代码生成

**提示词（Prompt）**：
```
我想用Python分析大学生消费数据，请给出一个pandas数据分析的模板代码，
包括：数据加载、清洗、描述性统计、可视化。
代码要简洁，有注释。
```

**AI响应摘要**：
```python
import pandas as pd
import matplotlib.pyplot as plt

# 加载数据
df = pd.read_csv('data.csv')

# 数据清洗
df = df.dropna()

# 描述性统计
print(df.describe())

# 可视化
df['消费金额'].hist()
plt.show()
```

**我的追问**：
```
如果有缺失值，我应该用什么方法填补？
```

**AI回应**：
- 数值型：用均值、中位数填充
- 分类型：用众数填充
- 高级：KNN或多重插补

**我的反思**：
> AI生成的代码框架很好，但我需要根据自己的数据特点调整清洗策略。
