## 迭代轨迹记录

### 问题1：Git提交失败

**问题描述**：
尝试commit时提示"nothing to commit"

**原因分析**：
文件已存在但未添加到暂存区

**解决过程**：
```bash
# 1. 检查状态
git status

# 2. 添加文件到暂存区
git add .

# 3. 再次检查
git status

# 4. 提交
git commit -m "描述信息"
```

**教训**：git add 是必须的，不能跳过

---

### 问题2：路径错误

**问题描述**：
创建文件时提示"路径不存在"

**原因分析**：
目录名称有中英文混合，路径解析出错

**解决过程**：
- 使用纯英文路径
- 避免在路径中使用空格和特殊字符
- 使用绝对路径确保准确

**教训**：保持路径简洁规范

---

### 问题3：AI协作中的提示词优化

**问题描述**：
AI给出的代码有语法错误

**原因分析**：
提示词不够具体，缺少上下文

**解决过程**：
1. 明确Python版本
2. 指定需要的库
3. 说明数据格式
4. 要求解释关键步骤

**优化后的提示词**：
```
我使用的是Python 3.14，需要pandas库处理CSV文件。
数据文件路径是 D:/Elite20/data.csv
请给出完整的、可运行的代码，包含错误处理。
```

**教训**：好的提示词 = 清晰的需求 + 具体的上下文

## 迭代方法论

### 验收标准优先

在动手之前，先写出"什么算成功"：

| 任务 | 验收标准 |
|------|---------|
| 数据分析报告 | 包含描述性统计+3个图表+1份报告 |

```
大任务 → 小步骤 → 每步验证 → 持续改进
```
