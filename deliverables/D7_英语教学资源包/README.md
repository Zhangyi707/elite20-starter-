## 🎯 设计理念

### K-S-T-A-R 应用

- **K（知识）**：大学英语教学大纲要求 + 学生实际水平
- **S（情境）**：大班授课，学生参与度低，个性化反馈不足
- **T（任务）**：制作可打印、可重复使用的教学工具
- **A（行动）**：使用CodeBuddy生成基础内容，教师审核调整
- **R（结果）**：获得可直接用于课堂的AI辅助教学资源

## 💡 使用建议

1. **词汇闪卡**：课前打印，课堂分组使用
2. **写作模板**：布置作业时提供框架支持
3. **讨论卡**：课堂热身环节，引发小组讨论
4. **文化工作表**：单元教学的文化背景拓展
