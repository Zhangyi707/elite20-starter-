## 📝 Template 1: Paragraph Structure

### Topic Sentence + Supporting Details + Concluding Sentence

```
[Topic Sentence - 主题句]
The main point of this paragraph is...

[Supporting Detail 1 - 支持细节1]
First, ...
For example, ...

[Supporting Detail 2 - 支持细节2]
Moreover, ...
In addition, ...

[Concluding Sentence - 总结句]
In conclusion, ...
Therefore, ...
```

**适用场景**：议论文正文段落

---

## 📝 Template 2: Opinion Essay

### 5-Paragraph Structure

```
【Introduction - 引言】
Hook: ... (引发读者兴趣)
Background: ... (提供背景信息)
Thesis Statement: ... (明确表达观点)

【Body Paragraph 1 - 正文1】
Topic Sentence: ... (第一个分论点)
Supporting Evidence: ...
Example: ...

【Body Paragraph 2 - 正文2】
Topic Sentence: ... (第二个分论点)
Supporting Evidence: ...
Example: ...

【Body Paragraph 3 - 正文3】
Topic Sentence: ... (第三个分论点)
Supporting Evidence: ...
Example: ...

【Conclusion - 结论】
Restate Thesis: ...
Summary: ...
Final Thought: ...
```

**适用场景**：四级/六级写作，雅思Task 2

---

## 📝 Template 3: Cause and Effect Essay

```
【Introduction】
Hook: ...
Background: ...
Thesis: ... (现象概述)

【Body - Causes 分析原因】
Cause 1: ...
  - Explanation: ...
  - Example: ...

Cause 2: ...
  - Explanation: ...
  - Example: ...

【Body - Effects 分析影响】
Effect 1: ...
  - Explanation: ...
  - Example: ...

Effect 2: ...
  - Explanation: ...
  - Example: ...

【Conclusion】
Summary: ...
So what? / Why does this matter? ...
```

**适用场景**：分析社会现象、讨论因果关系

---

## 📝 Template 4: Summary Writing

```
The passage primarily discusses...
The author argues that...
According to the passage, ...
The key point/finding is that...
It is also mentioned that...
Finally, the author concludes that...
```

**适用场景**：阅读理解简答题、摘要写作

---

## 📝 Template 5: Email Writing

```
Dear [Name],

【Opening - 开篇】
I am writing to... / I would like to... / Thank you for...

【Body - 正文】
1. ... (第一点)
2. ... (第二点)
3. ... (第三点)

【Closing - 结尾】
I look forward to hearing from you.
Please feel free to contact me if you have any questions.
Thank you for your time and consideration.

Best regards,
[Your Name]
```

**适用场景**：正式邮件沟通

## 💡 使用建议

1. **先模仿**：选择适合的模板，填入自己的内容
2. **再创造**：熟练后逐步形成个人风格
3. **AI辅助**：可以用AI帮你填入具体内容，但结构和逻辑需要自己把控
