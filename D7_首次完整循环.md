**步骤1：数据生成**
```python
import pandas as pd
import numpy as np

# 创建模拟的大学生消费数据
np.random.seed(42)
n = 100  # 100名学生
data = {
    '学生ID': range(1, n+1),
    '月生活费': np.random.randint(1000, 3000, n),
    '餐饮消费': np.random.randint(300, 1200, n),
    '购物消费': np.random.randint(100, 800, n),
    '娱乐消费': np.random.randint(50, 500, n),
    '学习消费': np.random.randint(50, 300, n)
}
df = pd.DataFrame(data)
df.to_csv('大学生消费数据.csv', index=False, encoding='utf-8-sig')
```

**步骤2：数据分析**
```python
# 描述性统计
print(df.describe())

# 计算消费占比
df['总消费'] = df['餐饮消费'] + df['购物消费'] + df['娱乐消费'] + df['学习消费']
df['餐饮占比'] = (df['餐饮消费'] / df['总消费'] * 100).round(1)
```

**步骤3：可视化**
- 生成柱状图：各消费类别平均占比
- 生成饼图：消费结构分布
- 生成趋势图：模拟月度消费变化

**步骤4：报告撰写**
使用CodeBuddy辅助撰写分析报告，补充结论和建议。
