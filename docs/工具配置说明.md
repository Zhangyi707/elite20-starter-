## 🔧 常用命令速查

### Git 基础命令

```bash
# 克隆仓库
git clone https://github.com/Zhangyi707/elite20-starter.git

# 查看状态
git status

# 添加文件到暂存区
git add .

# 提交
git commit -m "描述你的更改"

# 推送到远程
git push origin main

# 查看提交历史
git log --oneline
```

### Python 环境

```bash
# 检查Python版本
python --version

# 创建虚拟环境（可选）
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
```

## ❓ 常见问题

### Q: Git push 失败怎么办？

1. 检查GitHub认证是否正确配置
2. 确认remote地址正确：
   ```bash
   git remote -v
   ```
3. 如果使用SSH，确保SSH密钥已添加到GitHub

### Q: Claude/CodeBuddy 无法连接？

1. 检查网络连接
2. 确认工作区文件夹权限
3. 重启应用程序

### Q: 如何跟踪学习进度？

使用每日反思日志（templates/反思日志_模板.md）记录学习历程，并在D10准备3分钟演示。
