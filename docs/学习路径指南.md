## 🗺️ 辫状轨道模型

课程采用双轨并行设计，每天至少涉及两个轨道：

| 轨道 | 定位 | 类比 |
|:----:|------|------|
| T1 | 世界观：为什么AI+X重要 | 我们正在绘制的地图 |
| T2 | 环境：账户、工具、工作区 | 工作间 |
| T3 | GitHub：版本控制 | 实验室笔记本 |
| T4 | Skills：调用预构建技能 | 智能体阅读的食谱卡片 |
| T5 | 氛围编程：K-S-T-A-R循环 | 爵士乐，而非照谱演奏 |

## 📝 每日课程结构（60分钟）

```
[5分钟]  开场 - 今日主题 + 链接上一课
[10分钟] 讲解 - 核心概念（用类比）
[25分钟] 动手做 - 挑战任务
[15分钟] 反思 - K-S-T-A-R回顾
[5分钟]  收尾 - 明日预告
```

## 🎓 核心概念速查

### K-S-T-A-R 循环

| 字母 | 含义 | 问题引导 |
|:----:|------|----------|
| **K** | 知识 Knowledge | 我需要什么信息？ |
| **S** | 情境 Situation | 具体背景是什么？ |
| **T** | 任务 Task | 我真正要完成什么？ |
| **A** | 行动 Action | 我/AI做了什么？ |
| **R** | 结果 Result | 发生了什么？学到了什么？ |

### AI+X 三维框架

| 轴 | 含义 | 说明 |
|:--:|------|------|
| **X** | 领域 | 你的专业、语言、品味、关心的问题 |
| **AI** | 杠杆 | 提示词、Skills、MCP工具、智能体、工作流 |
| **S** | 自我 | 反思、作品集、K-S-T-A-R复盘 |
